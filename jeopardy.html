JEOPARDY

Jeopardy is a very popular gameshow in which contestants answer difficult trivia questions about certain topics presented for specific point values. This implementation of Jeopardy is very similar, but allows for variety for the creator and users. Users can make a board of whatever dimensions of their choosing, personally write each question and its respective answer, as well as assign custom point values to each question. The board is then generated and players can answers questions while having their score tracked between rounds.

This project was assigned in a Web Programming class I took at UVA, and the program itself ran on Java Servlets and Apache Tomcat was used to host it locally during playtime and demonstrations.
